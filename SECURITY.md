# Security Policy

If you discover a security issue in REscript, do not open a public issue with exploit details.

Report it privately to:

- [support@kevv.ai](mailto:support@kevv.ai)

Please include:

- a clear description of the issue
- affected routes, files, or flows
- reproduction steps
- expected impact
- any suggested mitigation if you have one

We will review reports privately and coordinate a fix before public disclosure when appropriate.
